# Pokédex Demo App

A modern Pokédex application built with React, Vite, Redux Toolkit, and Tailwind CSS. Discover and explore the world of Pokémon with detailed information, filtering, and search capabilities.

## Features

- **Browse Pokémon**: View all Pokémon with pagination
- **Search**: Find Pokémon by name or ID
- **Filter**: Filter by type and gender
- **Detailed View**: Click any Pokémon card to see detailed stats, abilities, and descriptions
- **Responsive Design**: Works on desktop and mobile devices
- **Loading States**: Smooth loading animations and states

## Tech Stack

- **React 19** - UI library
- **Vite** - Build tool and dev server
- **Redux Toolkit** - State management
- **Tailwind CSS** - Styling
- **PokéAPI** - Pokémon data source

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start development server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
